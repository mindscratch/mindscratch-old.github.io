Gobrew::fw

by Craig Wickesser - Nov 15th, 2013 - posted in golang,go

In the Ruby world there’s a tool called RVM which makes it easy to install and switch between multiple Ruby versions. However, for developers that work inside a restrictive LAN or behind a Firewall that prevents easy access to sites on the Internet, RVM wasn’t a viable option until RVM::FW was born. Steven Haddox created RVM::FW to allow developers to use RVM in restricted environments.

I’ve helped contribute (a little) to RVM::FW and I’ve also setup and managed RVM::FW in my work environment.

I’ve been slowly learning Go and just last night I stumbled onto gobrew:

Shell script to download and set GO environmental paths to allow multiple versions.

I believe there are few alternatives to gobrew such as gvm, but I found gobrew first and I’ve used it so I’m focusing on it (I’ve also just contributed to it).

Since RVM::FW has brought me a lot of joy at work and I’m a budding Go programmer, I’ve decided to take on gobrew::fw. The project has just been created so there’s nothing to see just yet, but soon enough I hope to have an application that enables:

  • being able to download and install gobrew
  • being able to be a source for Go installation files
    • instead of gobrew trying to download them from go.googlecode.com

Are you interested in this project? Would it be of use to you? If so, drop me a message on twitter, I’m @mind_scratch.
